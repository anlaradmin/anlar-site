# ANLAR — African Network for Learning Analytics Research

Official website for the African Network for Learning Analytics Research (ANLAR), a Special Interest Group of SoLAR.

Hosted on GitHub Pages at [www.anlar.org](https://www.anlar.org)

## Structure

```
index.html          — Single-page website
images/             — Logo and images
CNAME               — Custom domain configuration
```

## Editing

Edit `index.html` directly on GitHub or clone the repository locally. Changes pushed to the `main` branch are published automatically.
